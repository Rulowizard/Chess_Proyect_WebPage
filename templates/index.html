About


Previous Work

The present web page is the continuation of the previous chess project. That project focused on creating a program capable of playing a match of chess; from this point on we will refer to it as the "chess engine". The chess engine incorporated five machine players and a human interface. Depending on the version of the machine player, the engine would consider different variables to decide the optimal play. The most advanced version of the machine player incorporates a Mini-Max algorithm. The mini-max algorithm lets the machine player to "see" what the oponent will do and react to its actions.


Chess Website

The goal for the second part of the project was to create a web page capable of hosting the chess engine. The web page includes the following functionalities:

  • A flask application that will host the web page and control the flow of the chess game
  • A connection to MySQL and MongoDB
  • Web scrapping to obtain news related to chess
  • Graphics to show information about the games played
